Netcomm simple installer package for Weaved Target Device
---------------------------------------------------------

Installs Weaved on a Netcomm modem.

./install 

from this directory to install.


Copies weavedConnectd.arm-gnueabi-shared to /usr/bin/weavedConnectd

Create the directory /etc/weaved

Copies enablement files to /etc/weaved

Copies startup scripts to /etc/init.d
